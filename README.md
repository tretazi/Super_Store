#  Superstore Sales Dashboard

> Projet portfolio — Data Analyst | Excel + Python + Streamlit

##  Contexte
Analyse complète de 4 ans de données de ventes (9 994 commandes)
pour une chaîne de distribution, avec dashboard interactif.

##  Livrables
-  Nettoyage complet des données (rapport Excel)
-  Analyse CA, marge, croissance YoY
-  Analyse produit : Top/Flop, sous-catégories
-  Dashboard interactif Streamlit
-  Recommandations business

## 🛠️ Stack technique
- **Nettoyage** : Excel + Power Query
- **Dashboard** : Python, Streamlit, Plotly, Pandas

##  Lancer le dashboard en local
pip install -r requirements.txt
streamlit run app.py

##  Dataset
Superstore Sales Dataset — [Kaggle](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)
